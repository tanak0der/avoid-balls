avoid-balls
====

ボールを避けて進んでいくゲームです。  
APサーバにNode.jsをつかている以外はネイティブのJavascriptだけで作っています。

## Description

Javascriptの勉強用に作成。  
Window関数のrequestAnimationFrameを使用し、ゲームループの中でゲーム処理、座標計算、描画を行なっています。
